# Healthcare-Workflow-Dashboard
A detailed Power BI dashboard built using 7,000+ healthcare records to analyze patient workflow, diagnosis patterns, doctor feedback, and billing performance.

Highlights
- Tracks admit, discharge, and follow-up timelines.
- Breaks down bed occupancy across Private, General, and ICU.
- Shows diagnosis distribution and patient volume.
- Compares billing amounts with insurance coverage.
- Displays doctor feedback scores and patient interactions.
- Includes filters for date ranges and patient IDs.
